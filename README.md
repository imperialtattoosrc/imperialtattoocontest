# imperialtattoocontest
O projeto Imperial Tattoo Contest nasce da vontade de democratizar o acesso ao entretenimento e suprir a carência de eventos culturais na região de São Cristóvão.  Localizado no espaço Vento Solar, somos vizinhos da Rua Ceará e do icônico espaço 'Garage' — o berço histórico do rock carioca.
